Humidity vs Latitude



The program selected a random set of 1500 latitude and longitude measurements, from which we mapped to the city nearest these coordinates (and then eliminated any repeat cities from the dataset). Similar to the Cloudiness vs Latitude graphic, there doesn't appear to be much correlation between Humidity % and Latitude. Perhaps a data anomaly, but you will note the absence of a 0% humidity data points at the extreme northern latitudes, while there are 0% humidity measurements seen at the extreme southern latitudes.
